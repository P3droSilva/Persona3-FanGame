# Persona 3 FanGame
Uma tentativa de replicar aspectos básicos do jogo Persona 3 como um trabalho para a disciplina de Computação Gráfica Avançada da UFSM em 2023/2.

Desenvolvido em Unity, em um intervalo de tempo curto.
